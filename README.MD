# eslint-config-air-base

Airbnb style guide without semicolons and React plugins

## How To Use

Install the package
```Shell
npm add -D eslint-config-air-base
```

Install peer dependencies if you haven't yet

```Shell
npm add -D eslint
```

Add extends property to an ESLint config file

```JS
extends: 'air-base',
```
